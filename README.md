## Spartist

The Foodio web app was made with React. Similar to social media apps, this web app uses React to provide the user with a timeline of fake cooking recipes.

Foodio is created using React.

## INSTALLATION

SSH

```bash
git clone git@github.com:MichelleW31/Foodio.git
```

HTTPS

```bash
git clone https://github.com/MichelleW31/Foodio.git
```

Use the package manager npm to install Foodio.

```bash
npm install
```

The Spartist app can also be seen on https://michellew31.github.io/Foodio/
